# Art Project

## Description
This is the revised version of a website I originally built in 2020 for a client who wanted to showcase artwork produced during the coronavirus lockdown.  

The updated site features a modern, sleek design while highlighting creative works such as paintings, music, crafting, and a charity bike run. I implemented JavaScript for a functional hamburger menu to enhance navigation, and used Flexbox to improve layout responsiveness across different screen sizes.

The website demonstrates both my early and current web development skills, combining functional design with improved aesthetics and user experience.

## Technologies Used
- HTML5
- CSS3
- JavaScript
